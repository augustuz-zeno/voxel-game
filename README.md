# Воксельная Игра (Voxel Game)

Минималистичный клон Minecraft, написанный на Rust с использованием игрового движка [Bevy 0.14](https://bevyengine.org/). 

> **Внимание:** Это максимально сырая версия игры, находящаяся на раннем этапе разработки. Многие функции, такие как инвентарь, крафт, генерация биомов, ломание и установка блоков, пока не реализованы.

## Возможности на данный момент
- Процедурная генерация бесконечного ландшафта (холмы, вода, песок, земля, камень).
- Базовая физика и коллизия (гравитация, прыжки, столкновения со стенами).
- Режим полёта (креатив) и режим выживания.
- Встроенный дебаг-оверлей с основной статистикой.

## Управление

* **WASD** — Перемещение
* **Space (Пробел)** — Прыжок. Если зажать, игрок будет автоматически прыгать при касании земли.
* **Двойное нажатие Space** — Переключение между режимом полёта (Creative) и обычной ходьбой (Survival)
* **Shift (в режиме полёта)** — Спуститься вниз
* **Ctrl (Левый)** — Ускорение (Бег)
* **F3** — Показать/скрыть дебаг-оверлей (ФПС, время кадра, координаты, количество загруженных чанков)
* **ЛКМ** — Захватить курсор
* **Esc** — Освободить курсор (повторное нажатие — выход в главное меню)

## Сборка и запуск

Убедитесь, что у вас установлен Rust и Cargo.

```bash
cargo run --release
```

(Сборка в release-режиме рекомендуется для максимальной производительности).
